# Read me
F# syntax highlighting for Sublime Text.

Based on [TextMate FSharp bundle][1] version 0.5. Changes/Fixes are more then
welcome!

Including a sublime-build; you need to add the path to 'fsi.exe' or 'fsharpi' in your $PATH
